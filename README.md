# Django Blog Project

A simple blog web application built with Django. This project allows users to create, view, and manage blog posts. It is currently a work in progress as more features and improvements are being implemented. 

## Features

Currently implemented:

- Django project setup
- Blog application structure
- Blog post model

Work in progress:

- Database integration using SQLite
- Admin dashboard for managing posts
- Displaying blog posts
- User authentication
- Creating posts from the frontend
- Editing and deleting posts
- Comments system
- Styling and UI improvements
- Deployment

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Git & GitHub
